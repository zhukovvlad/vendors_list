# Девлог

Журнал выполнения задач по проекту (учёт вендор-листов и соответствия
стандартам). Хронология работ — фиксируем решения, их причины и подводные
камни, чтобы контекст не терялся между сессиями.

Каждый отчёт — отдельный файл вида `YYYY-MM-DD-<краткое-название>.md`.
Структура отчёта: что сделано, верификация, решения и нюансы, что осталось.
Запись добавляется после мерджа задачи в `main`.
